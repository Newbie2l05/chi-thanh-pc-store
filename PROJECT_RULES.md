# 13. Quy tắc đọc file và tối ưu context

AI không được quét hoặc đọc toàn bộ mã nguồn WordPress nếu không cần thiết.

Chỉ đọc những thư mục và file liên quan trực tiếp đến tác vụ hiện tại.

Ưu tiên:

* wp-content/themes
* wp-content/plugins
* file giao diện đang được sử dụng
* CSS
* JS
* template WooCommerce liên quan

Không đọc toàn bộ:

* wp-admin
* wp-includes
* uploads
* cache
* logs
* node_modules
* vendor

Khi thực hiện tác vụ:

1. Xác định file liên quan trước
2. Chỉ đọc các file cần thiết
3. Giải thích file nào sẽ được đọc
4. Hạn chế tiêu tốn context/token

Luôn ưu tiên:

* hiệu năng
* tối ưu token
* tránh scan toàn bộ project

Không được rewrite toàn bộ theme hoặc toàn bộ project nếu chưa được cho phép.
