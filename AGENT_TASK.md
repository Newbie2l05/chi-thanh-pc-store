## Mục tiêu hiện tại

Xây dựng website WordPress + WooCommerce bán linh kiện PC/gaming gear với giao diện dark premium lấy cảm hứng từ các website công nghệ cao cấp.

Không copy logo, hình ảnh, nội dung hoặc layout y nguyên của thương hiệu thật.

## Việc cần làm trước

1. Phân tích project hiện tại.
2. Xác định theme/plugin đang dùng.
3. Chỉ đọc các file liên quan trong:
   - wp-content/themes
   - wp-content/plugins
4. Không đọc toàn bộ wp-admin, wp-includes, uploads, cache.
5. Lập kế hoạch thực hiện.
6. Hỏi tôi xác nhận trước khi sửa code lớn.

## Ưu tiên

1. Làm giao diện trang chủ thật đẹp.
2. Tạo cấu trúc WooCommerce chuẩn.
3. Tạo plugin PC Builder Compatibility.
4. Tạo shortcode demo được.
5. Đảm bảo dễ deploy và đẩy GitHub.
